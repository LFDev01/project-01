- SITE DE PRODUTOS DE TECNOLOGIA - 

1. Página Principal
    1.1 -  A página principal conterá uma breve seção sobre a empresa e um menu de navegação que levará o usuário para outras páginas.

    1.2 - Este menu será formado pelas seguintes opções: Produtos, ???, Suporte (Contato) e uma opção para entrar na conta do usuário.
        1.2.1 - A opção de produtos será um menu dropdown que mostrará as categorias de produtos vendidos pela empresa. Cada opção de produto levará o usuário para uma página que contém produtos da categoria selecionada.


2. Página de Produtos
    2.1 - A página de produtos será constituída de 4 colunas do produto da categoria selecionada pelo usuário.

    2.2 - Cada coluna possuirá os seguintes items:
        2.2.1 - Uma imagem que se expande até o tamanho máximo de seu container.
        2.2.2 - Acima da imagem, estará o nome do produto.
        2.2.3 - Logo abaixo da imagem, haverá o preço do produto.
        2.2.4 - Abaixo do preço do produto, haverá dois items: Um botão com ícone de carrinho de compra escrito ao lado "comprar", e em seguida outro botão de especificações técnicas do produto. Esses botões devem estar lado a lado.

    2.3 - A especificação do layout acima deve se repetir para todas as categorias de produtos.
        2.3.1 - Isso significa que a página de smartphones e laptops devem ter exatamente a mesma estrutura.


3. Página de Login
    3.1 Um container centralizado de duas colunas. Esse container conterá:
        3.1.1 - Uma imagem ilustrativa.
        3.1.2 - Ao lado da imagem, um formulário solicitando informações de login: E-mail e Senha.
        3.1.3 - Botão de login abaixo do formulário.
        3.1.4 - Um link de cadastro abaixo do botão de login para caso o usuário não possua uma conta.
